## File Menu ##
| **Action** | **Shortcut** |
|:-----------|:-------------|
| New Window | Ctrl+N |
| New Tab | Ctrl+T |
| Open File | Ctrl+O |
| Open Location | Ctrl+L |
| Close Tab | Ctrl+W or Ctrl+F4 |
| Save As... | Ctrl+S |
| Print | Ctrl+P |
| Close Window | Ctrl+Shift+W |
| Quit | Ctrl+Q |
